Automatic rigging script

Fall 2016

I am currently working on an automatic rigging script in Maya, using the Maya python API. My goal is to create a script that automatically generates controls for human characters.

Here are some in progress shots. The model was generated using Mixamo, the UI was created using PySide. The script takes in the names for each joint and rigs the model according to the joints, so it assumes that certain joints are present in the model.


FITYMI Animation

Spring 2016

I worked on this project as a technical artist. The film “Lovers Meet” is a blended live-action and 3D animated short film about a young man, Isaac, who finds himself inside the world of an impressionist painting.

My main task was retargeting and refining mocap data. I also worked on modeling, rigging, camera setup, and lighting, using Autodesk Maya and MotionBuilder. Below is an example of a scene that I worked on.


Below is an example of how I refined the mocap animation in MotionBuilder. I first get the raw mocap data, and retarget the animation to the models. Then, I adjust the keyframes of the animation to get the final result.

Fluid Simulation

Fall 2013

Real-time fluid simulation in CUDA using the SPH algorithm, which involved dividing the particles up into grids and doing the calculations based on how the neighboring grids and particles interact with each other. The force on each particle will then be calculated depending on it's neighbors.

Raytracer

Fall 2012

Wrote a raytracer using C++ that supports reflections and refractions. Raytracing is a rendering technique that traces a ray through a scene to produce a 3D image. The reflection and refraction rays are done by recursively casting rays from their intersection points.

Arduino projects

Fall 2012

Several small projects made with an Arduino.

Hallowuino

Horrifying ghoul made for Halloween.There is a photoresistor in its nose, which detects the amount of light in the environment. When the lights are off, the servos twitch at random angles to move the eyes and mouth.


Cat vs Bird

Automaton project. I went through several prototypes before settling on this setup, mainly because I spent most of the time figuring out how the automino kit works and looking up various rack and pinion setups online.


Pavlov's M&Ms

In the game, you move a small square with your mouse, and click to shoot. If you shoot a big M&M, a smaller M&M will fall from it. The goal of the game is to collect as many small M&Ms as possible, and the machine will dispense an M&M when you beat the high score.
